# Weather-Insights-PowerBI-Dashboard
📌 Overview

Built an interactive Power BI dashboard using live Weather API data to monitor temperature trends, AQI, humidity, wind speed, rainfall probability, and multi-day forecasts. The project focuses on data extraction, transformation, modeling, and real-time visualization.

🛠 Tech Stack

Power BI, Weather API, Power Query, DAX

🔄 Workflow

Extracted live API data

Cleaned and transformed data using Power Query

Created calculated measures and KPIs using DAX

Designed dynamic visualizations with automated refresh

📊 Key Features

Temperature trend analysis

Air Quality Index monitoring

Rain probability insights

KPI cards for real-time weather metrics
